1. Welcome

Sit-O-CHIRARI is a nanjaku (means 'soften & weaken' in Japanese) desktop accessory. Small Japanimation girl bitmap named 'Serina' will sit upon current active window.
  • Her cute pose, with low-angle :)
  • She sometimes winks :)
  • She can check POP mailbox arrival
  • Changable her appearence by choosing pane files
  • You can associate MMPM/2 multimedia files to some events
  • 'Nanjaku OS/2 Project'-ready, OS/2 native, fully 32-bit, PM multithreaded nanjaku application :)
Please tell me your impressions / requests :)

1.2. System Requirements

  1. This program will be functioned in environment such as:
    • Presentation Manager on OS/2 version 4.0, WSeB, MCP, ACP or eCS, applying latest FixPak recommended
    • 256+ colored (65536+ recommended) display adapter / PM driver configuration
    POP mailbox checking requires that:
    • TCP/IP for OS/2 version 4 or later, applying latest FixPak recommended
    • Valid POP mailbox account :)
    MMPM/2 multimedia playing of course requires MMPM/2 feature installation.
  2. Currently, There is a problem that this program will cause PM freeze together with Java 1.3 window. We recommend not to use this program when any Java 1.3 applications are launched.
    Regretfully, no solvable actions in this program.

2. How to use

2.1. Basic Instructions

  1. To launch, open 'Sit-O-CHIRARI' WPS program object in same name folder. At the first execution, 'Product Information' dialog will be displayed.
    Image of 'Product Information' dialog
  2. In default, 'Serina' sits upon either top edge of current active frame window or rim of bottom one, at leftmost. Sorry, she cannot be dragged nor moved to directly.
    Image of standard pane 'Serina the schoolgirl'
  3. Messages from her will be shown in right cartoon baloon. After a few seconds, this baloon will be hidden automatically.
  4. In order to change font face / size of her message, drag appropriate font from WPS Font Palette and drop it to cartoon baloon while baloon shows :)
  5. To close her,
    • Click right mouse button on 'Sit-O-CHIRARI' window (popup menu will be shown), then choose 'Close'.
    • Use 'Window List' :)

2.2. Advanced Instructions

  1. Clicking right mouse button on 'Sit-O-CHIRARI' window shows popup menu.
    Properties...
    Configures preferences of 'Sit-O-CHIRARI'. See below.
    Panes
    Listing of available panes in submenu. Checked item is current pane.
    Choosing submenu item will change appearence immediately. Baloon messages will be changed also.
    Some followed panes are included:
    'Serina the maidservant' pane (MAID.pnl)
    Just a title, maidservant costume :)
    Image of 'Serina the maidservant' pane
    Check mailbox now
    Checks POP mailbox arrival immediately. 'Mailbox check' settings are required. See below.
    Chastise now!
    Shows message suitable for your choice :) For stress alleviation...
    Help
    Displays some helpful informations.
    User's guide
    Displays User's guide (this document), using Web brouser associated to WPS URL object.
    Product information...
    Displays 'Product Information' dialog.
    Close
    Closes 'Sit-O-CHIRARI'.
  2. In order to modify 'Sit-O-CHIRARI' preferences, choose 'Properties...' of popup menu. Changes will be applied by 'Apply' button. Pressing 'Cancel' throws any changes away.
    'General' page
    General settings.
    Double click action
    Specifies issueing action by double-clicking her. Choose one from 'None', 'Check mailbox now' or 'Chastise now!'.
    Process priority boost
    Specifies whether boosting process execution or not. More smooth drawing / movement, but may stress your system.
    Mouse event transparency
    Specifies whether passing any mouse events through 'Sit-O-CHIRARI' or not. Useful if you feel your mouse operations are hindered by her.
    To show popup menu if enabled (or any mouse actions such as double-click or drag & drop), do with pressing both Shift and Ctrl keys.
    Reset font setting
    If checked, current font setting for cartoon baloon messages will be reset to system default.
    'Sitting' page
    Settings releted to her sitting position.
    Horizontal position
    Specifies her horizontal position in percent, with relative ratio of sat frame window's width. 0% corresponds to leftmost and 100% corresponds to rightmost.
    Vertical position
    Specifies her vertical position alignment you want. Choose one from 'On top edge', 'On bottom edge' or 'Center-aligned'.
    'Mailbox check' page
    Settings in order to check POP mailbox arrival. Checking result will be shown in cartoon baloon.
    Enable POP mailbox checking
    If checked, mailbox checking becomes enable. Cannot be checked in non-TCP/IP-aware enironments.
    POP server
    Specifies valid address and port number of POP mailbox service server. Port number of 0 implies 110, POP service default.
    POP user and password
    Specified valid user name and password of checking POP mailbox account.
    Mailbox checking interval
    Specifies mailbox checking interval in minutes. 0 implies no automatic checking.
    Recaptioning with
    Specifies recaptioning method of hidden baloon. Choose from either 'Clicking mouse1 button' or 'Bluring mouse pointer'.
    'MMPM/2' page
    Settings in order to associate MMPM/2 multimedia playing to some events.
    Enable MMPM/2 multimedia play
    If checked, multimedia playing becomes enable. Cannot be checked in non-MMPM/2-aware environments.
    Mail absent
    Specifies multimedia file that corresponds to absent of arrival, as result of POP mailbox checking.
    Mail present
    Specifies multimedia file that corresponds to present one or more of arrival, as result of POP mailbox checking.
    Mail error
    Specifies multimedia file that corresponds to any errors occured, as result of POP mailbox checking.
    Chastising
    Specifies multimedia file that corresponds to issueing 'Chastise now!' action :)
    Pressing right button of each entryfields will open standard file dialog.
    In order not to associate file to specified event, leave correspond item empty.
